[Project Example] What is SELFRec?

SELFRec is a Python framework for self-supervised recommendation (SSR) which integrates commonly used datasets and metrics, and implements many state-of-the-art SSR models. SELFRec has a lightweight architecture and provides user-friendly interfaces. It can facilitate model implementation and evaluation.

Easy configuration

SELFRec provides simple and high-level interfaces, making it easy to add new SSR models in a plug-and-play fashion..

Highly Modularized

SELFRec is divided into multiple discrete and independent modules. This design decouples model design from other procedures, allowing users to focus on the logic of their method and streamlining development.
